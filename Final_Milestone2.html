Project Goal: The goal of this project is to investigate and determine the impact of environmental and weather conditions on athletic performance in the NFL based on betting odds and weather data from 2000-2020. To understand betting odds, visit this page, Betting Odds Explanation.

Description of Datasets:

Dataset 1: This Kaggle dataset contains comprehensive weather data for NFL games 2000 - 2020 from MeteoStat and Weather Underground. This dataset in conjunction with Project Dataset 2 will be the data we use to analyze the influence of weather conditions on athletic performance in the NFL. It compromises 3 primary data tables: (games_2.csv, games_weather.csv, and stadium_coordinates.csv).

games_2.csv: This table catalogs individual NFL games from 2000-2020, each represented by a unique game_id. For each game, it details the season and the stadium where it took place. This information is important for correlating specific games with corresponding weather data.

games_weather.csv: This table provides the weather observations associated with each game, linked by their game_id. It includes important weather parameters such as temperature, dew point, humidity, precipitation, and wind speed. These key weather matrics will enable an in-depth analysis of how specific weather conditions may have impacted game outcomes and player performances.

stadium_coordinates.csv: This table provides structural information about NFL stadiums. For each stadium, it specifies the roof type which is essential for accurately mapping the weather exposure during games.

Using this selected dataset, we can explore future research questions such as: Do teams with indoor stadiums or certain roof types perform more consistently in adverse weather conditions?, Which stadiums experience the most extreme weather variations, and does this create a meaningful home-field advantage?, Which teams have the most warm-weather games?, Which teams have the most cold-weather games? and more.

Dataset 2: This second dataset, games.csv, located at Git Hub compiles comprehensive NFL game data for analysis and research. Each row represents a single game with a unique game_id and includes several key variables for analyzing the impact of various environmental/weather factors on NFL game outcomes. The inclusion of variables like temperature, wind speed, roof type, and playing surface allows for an in-depth exploration of environmental influences on athletic performance. Additionally, betting-related variables such as spread_line and total_line provide insights into pre-game expectations versus actual results. Using this dataset, we can explore research questions such as: How do different weather conditions (temperature, wind speed) correlate with game outcomes and pre-game betting odds?, How does wind speed affect passing versus rushing performance in NFL games?, Do teams playing on artificial turf perform differently in adverse weather conditions compared to those playing on grass?, Are games played in domed stadiums more likely to meet or exceed betting expectations compared to those played in open-air stadiums?, and more.

from sklearn.model_selection import train_test_split
from sklearn.pipeline import make_pipeline
from sklearn.impute import SimpleImputer
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import confusion_matrix, ConfusionMatrixDisplay

kaggle_map = dict(
    zip(
        df_games_2['game_id'].astype(str),
        df_games_2['stadium_name']
    )
)
df_weather['stadium'] = df_weather['game_id'].astype(str).map(kaggle_map)

stadium_humidity = (
    df_weather
      .dropna(subset=['stadium'])
      .groupby('stadium')['Humidity']
      .mean()
)

df_humidity = (
    stadium_humidity
      .reset_index()
      .rename(columns={'Humidity':'avg_humidity'})
)

df = df_games.merge(df_humidity, on='stadium', how='left')

df['underdog_is_away'] = df['away_moneyline'] > df['home_moneyline']
df['underdog_is_home'] = ~df['underdog_is_away']
df['underdog_win'] = np.where(
    df['underdog_is_away'],
    df['away_score'] > df['home_score'],
    df['home_score'] > df['away_score']
).astype(int)

X = df[['temp', 'wind', 'avg_humidity', 'underdog_is_home', 'spread_line']]
y = df['underdog_win']

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42, stratify=y
)

pipe = make_pipeline(
    SimpleImputer(strategy='mean'),
    LogisticRegression(max_iter=1000)
)
pipe.fit(X_train, y_train)

train_acc = pipe.score(X_train, y_train)
test_acc  = pipe.score(X_test, y_test)
print(f"Train Accuracy: {train_acc:.3f}")
print(f" Test Accuracy: {test_acc:.3f}\n")

y_scores = pipe.predict_proba(X_test)[:, 1]
y_pred = pipe.predict(X_test)

order = np.argsort(y_scores)
y_scores_sorted = y_scores[order]
y_true_sorted = y_test.values[order]
y_pred_sorted = y_pred[order]
correct_mask = (y_pred_sorted == y_true_sorted)

plt.plot(y_scores_sorted, linewidth=1.5, label='Predicted P(underdog win)')
plt.scatter(
    np.where(correct_mask)[0],
    y_scores_sorted[correct_mask],
    c='blue', marker='o', label='Correct'
)
plt.scatter(
    np.where(~correct_mask)[0],
    y_scores_sorted[~correct_mask],
    c='red',  marker='x', label='Incorrect'
)
plt.xlabel('Test samples (sorted by predicted P)')
plt.ylabel('Predicted probability')
plt.title('Model Predictions & Correctness')
plt.legend(loc='lower right')
plt.grid(True)

from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

df_games["point_differential"] = df_games["home_score"] - df_games["away_score"]
features = ["temp", "wind", "spread_line", "surface", "point_differential"]
df_model = df_games[features].replace(-10000, pd.NA).dropna()
df_model = pd.get_dummies(df_model, columns=["surface"], drop_first=True)

X = df_model.drop(columns="point_differential")
y = df_model["point_differential"]
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=1)

model = LinearRegression()
model.fit(X_train, y_train)

y_pred = model.predict(X_test)
print("Coefficients:", dict(zip(X.columns, model.coef_)))
print("Intercept:", model.intercept_)
print("Mean Squared Error:", mean_squared_error(y_test, y_pred))
print("R^2 Score:", r2_score(y_test, y_pred))

m, b = np.polyfit(y_test, y_pred, 1)

errors = np.abs(y_pred - y_test)
median_err = np.median(errors)
correct = errors <= median_err
